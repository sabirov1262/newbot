RENDER SOZLASH

1) Render'da yangi service oching yoki hozirgisini Worker qilib yarating.
2) Type: Background Worker
3) Build Command: pip install -r requirements.txt
4) Start Command: python bot.py
5) Environment Variables:
   BOT_TOKEN = sizning bot tokeningiz
   SUPER_ADMIN_ID = sizning telegram id raqamingiz
6) runtime.txt avtomatik Python 3.11.9 ishlatadi.

MUHIM:
- Web Service emas, Background Worker bo'lishi kerak.
- Chunki bu bot polling ishlatadi va port ochmaydi.
